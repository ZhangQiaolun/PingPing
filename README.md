## Git代码部署说明
### 提交到新浪云

```
# 在你应用的git代码目录里，添加一个新的git远程仓库 sae
$ git remote add sae https://git.sinacloud.com/aipingping(已不用操作）

# 编辑代码并将代码部署到 `sae` 的版本2。
$ git add .
$ git commit -m 'Init my first app'
$ git push sae master:2
```
### 提交代码到github
$ git push
